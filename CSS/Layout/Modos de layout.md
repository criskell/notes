# Modos de layout

- Um modo de layout, também chamado de layout, é um algoritmo que determina os **tamanhos** e as **posições** de **caixas** em relação as caixas irmãs e ancrestrais.
- Alguns modos de layout:
  - Fluxo normal
  - Tabelas
  - Flutuado
  - Posicionado
  - Multicolunas
  - Flexbox
  - Grid
